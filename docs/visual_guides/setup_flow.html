FlowBridge Setup — 7 Steps, 25 Minutes

From zero to auto-forwarding jobs. No coding required.

1

📊 Create Google Sheet

2 min Open sheets.google.com, click "+ Blank", rename to "FlowBridge Queue". Copy the Sheet ID from URL.

2

☁️ Set Up Google Cloud OAuth

10 min console.cloud.google.com → create project "flowbridge" → enable Sheets + Drive + Gmail APIs → create OAuth Desktop Client → download credentials.json.

3

📥 Download FlowBridge

2 min git clone the repo OR download ZIP from GitHub. Extract.

4

🔑 Place credentials.json

1 min Move downloaded credentials.json into flowbridge/setup/ folder.

5

🚀 Run Setup Script

3 min Windows: .\scripts\setup-windows.ps1 — Mac/Linux: bash scripts/setup-macos.sh. Paste Sheet ID when prompted. Docker builds and starts.

6

📱 Scan WhatsApp QR

1 min Open VNC viewer → connect to localhost:5900 → see Chrome with QR → scan from phone (Linked Devices). Done forever.

7

📋 Set Up Sheet Tabs

2 min Add "Queue" tab headers (9 columns) + "Groups" tab with your WhatsApp group names. Agent polls every hour from here on.
